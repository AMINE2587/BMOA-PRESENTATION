BMOA Presentation

Innovative Digital Banking Solution

Developed by student Mohammed Amine El Barnoussi

The General Idea of ​​The Site

This project represents a simplified digital banking platform inspired by modern services like Payoneer. It simulates the user experience of real-world banking applications by offering features such as user authentication, a personalized dashboard, and the ability to view and manage financial data. The platform also includes virtual card functionality and account update options, providing users with a clean, responsive interface that reflects the core features of today’s digital banking solutions.

Techniques and Tools Used

This project was built using modern web development tools, including HTML for structure, CSS for styling,

and JavaScript for interactivity. JavaScript files are organized for better code management,

and GitHub is used for version control and collaboration.

Login/Sign Up

This is a Login and Sign-Up program built with HTML, CSS, and JavaScript.
It uses localStorage to save user data like email and password, allowing
users to sign up and log in without a backend. The design is modern,
with a clean UI.

Dashboard

This is the BMOA Dashboard interface, designed with a modern tech.
It provides users with an overview of their account after logging in.
The dashboard includes a profile section, easy navigation, and clean visuals. It's built using HTML, CSS, and JavaScript, and is connected to the login system using localStorage, so user data is dynamically displayed after authentication.

Profile

The profile page displays the user's information retrieved from localStorage after login. It features a modern design and allows the user to change their profile picture, providing a more personalized experience.
